# binge

## Installation for dev
Simply copy below comands one after one.

Download project:
```
git clone git@github.com:mateuszbaranczyk/binge.git
```
Create virtual env:
```
python3 -m venv binge-env
```
Activate env:
```
source binge-env/bin/activate
```
Install packages:
```
pip install -r requirements.txt  
```
